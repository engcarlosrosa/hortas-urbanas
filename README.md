# 🌱 Hortas Urbanas V2
**Conectando Sustentabilidade Comunitária e Capital Privado**

O **Hortas Urbanas** é uma plataforma focada em democratizar o acesso à agricultura urbana, utilizando tecnologia para monitorar impactos socioambientais e atrair investimentos **ESG**.

### 🏆 Destaques da Versão Atual
* **Liderança Técnica:** Projeto liderado por Carlos Eduardo, com **98% do desenvolvimento original** de autoria própria.
* **Impacto Real:** Focado em métricas de sequestro de CO2, segurança alimentar e valorização imobiliária (até 5% em imóveis no entorno).
* **Arquitetura:** Desenvolvido em React.js com Firebase Firestore.

---

## 🚀 Visão do Projeto
Diferente da V1, a **V2** busca integrar o setor privado através de:
1. **Métricas de Carbono:** Conversão de biomassa produzida em crédito de carbono evitado.
2. **Dashboard para Patrocinadores:** Transparência total para empresas que financiam hortas específicas.
3. **Engajamento Comunitário:** Gamificação e rede social para hortelões urbanos.
