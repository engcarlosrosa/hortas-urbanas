# 👨‍💻 Guia de Contribuição Técnica

Bem-vindo(a) ao time de desenvolvimento! Siga os passos abaixo para garantir que seu ambiente esteja configurado corretamente e que suas contribuições sejam registradas com sua identidade acadêmica.

## 1. Configuração do Ambiente
* **Node.js:** Versão LTS recomendada.
* **Git & VS Code.**

## 2. Passo a Passo
1. **Clone o repositório:**
   `git clone https://github.com/engcarlosrosa/hortas-urbanas.git`
2. **Identidade Acadêmica (Obrigatório):**
   Execute dentro da pasta do projeto:
   `git config --local user.name "Seu Nome Completo (Acadêmico)"`
   `git config --local user.email "seu-email@faculdade.edu.br"`
3. **Instalação:**
   `npm install`
4. **Execução:**
   `npm run dev`

## 📂 Estrutura de Pastas
* `src/components/`: UI Kit e componentes reutilizáveis.
* `src/pages/`: Páginas principais (Home, Dashboard, Mapa).
* `src/services/`: Conexão com Firebase.
